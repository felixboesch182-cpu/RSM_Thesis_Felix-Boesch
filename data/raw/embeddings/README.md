# `data/raw/embeddings/` — PaECTER patent embeddings

Pre-computed PaECTER embeddings for the EPO and USPTO patent populations. These files are **not**
generated by the pipeline and are excluded from version control (see `.gitignore`). Download them
and place them in this folder before running Stage 05.

## Download

Pre-computed PaECTER embeddings (MPG Edmond repository):

<https://edmond.mpg.de/dataset.xhtml?persistentId=doi:10.17617/3.BGRPMI>

## Expected files

| File | Contents |
|---|---|
| `EPO_PaECTER_embeddings.parquet` | PaECTER embeddings for EPO patent families |
| `USPTO_PaECTER_embeddings.parquet` | PaECTER embeddings for USPTO patent families |
| `patent_features_base.parquet` | Focal-sample embeddings + priority dates |

First used by Stage 05 notebooks `01`, `03`, `04`, `06`, `10`. The model itself
(`mpi-inno-comp/paecter`) is downloaded from Hugging Face on first use; only the pre-computed
embedding tables above need to be placed here manually.
